# SimulRepile

Ce dépôt fournit les deux firmwares nécessaires à l'architecture bi-carte de SimulRepile :

- `firmware/` : terminal graphique (Waveshare ESP32-S3-Touch-LCD-7B) avec LVGL/LovyanGFX.
- `core_firmware/` : cœur applicatif (ESP32-S3-DevKitC-1 WROOM-2) qui pilote la simulation et alimente l'afficheur via UART.

Consultez les README respectifs pour les instructions de build, le câblage et les détails d'intégration.
